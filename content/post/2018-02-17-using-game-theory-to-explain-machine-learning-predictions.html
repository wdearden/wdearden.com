In this post, I explain the Shapley value from cooperative machine learning and how it can be used to explain machine learning predictions.

The Shapley Value

Let’s say you and your friends Mimi, Colter, and Lance 1 are splitting an Uber. You live closest to the origin. Mimi and Colter live together just a few blocks farther and Lance lives the farthest away. Lance suggests that Uber has a handy built-in fare splitter which will split the fare evenly between all of you
